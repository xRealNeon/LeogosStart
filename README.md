# LeogosStart

Just a page with the logo of a search bar and forwarding options

## Arguments
| name   | value                                      |
| ------ | ------------------------------------------ |
| engine | google, duckduckgo, bing, ecosia           |
| curl   | for example: https://www.ecosia.org/search |
| cparam | for example: q                             |
| cname  | for example: Ecosia                        |
